ChatGPT & Twitter Bot Hosted on GCP

Consumer Secret/Key & Access Token can be obtained by switching to a twitter developer account. By importing the library tweepy it is straight forward to authenticate.
Trends are based on location so you need to use WOEID (Where On Earth ID). I have selected WOEID=23424848 which is New York City.



Similar as for Twitter, you can get the Access Key by creating a developer account on OpenAI. There are different models that can be selected here. To consider are the number of tokens to keep the free trial. As a rule of thumb a token are 4 characters or 0.75 words. For this reason I set the maximum of tokens to 52 so I get a short tweet.



The easiest way to get the script running is by creating a Virtual Instance under the Compute Engine Category from GCP. Keeping the instance running constantly is expensive and innecesary. For this reason I created a Task to wake up the instance at 11:55 and shut it down at 12:05. The cronjob that calls the script will run at 12:00.
