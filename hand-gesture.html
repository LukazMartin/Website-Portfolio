Scan: Selects all the text("Ctrl+A") and copies it to clipboard("Ctrl+C"). Now that it is on the clipboard, it identifies the names of the atributes within the function parameters. Max 2 attributes are allowed. One is saved as Variable a and the other as Variable b.

Hashmap: If hashmap exists, use it; else, create one.



For loop / if / return: Are predefined. Once choosen one, the code will know if tabs or spaces are needed.

Value/pos from for loop: The for loop will contain pos and val written as follows:for pos,val in enumerate(variable): pos and val can be choosen be clicking once or twice.



[]: Opening or closing brackets.

, / -: A comma or a substraction symbol.

= / left arrow: Equal symbol or 1 tab to the left (in case of wrong tabulation).

Ctrl+Z: In case of mistake.
